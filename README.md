# analysis
 筛选低估值的股票
 
 - 查询指定股票的滚动市盈率，根据PE/PB/市值的高低判断股票的估值高低
 - PE/PB/市值参考根据个股历史低估平均值+经验
 - 向指定邮件地址发送程序运行结果
 - 支持github的Actions功能，根据固定时间自动化执行程序
